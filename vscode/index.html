What is Visual Studio Code?

Visual Studio Code 是微软开发的免费开源代码编辑器，支持几乎所有编程语言的智能提示、调试和 Git 集成。丰富的扩展生态可变成 IDE、Markdown 编辑器或 Jupyter Notebook 环境。

✅ Why Download from the Official Source?

第三方网站常提供捆绑了广告插件或修改版。官方版本由微软数字签名，支持自动更新，无任何捆绑。

📖 Installation Guide

🪟 Windows: 前往 code.visualstudio.com/download → 选择 Windows → 运行 VSCodeSetup.exe → 建议勾选「添加到 PATH」。
🍎 macOS: 下载 macOS 版 → 解压 .zip → 将 Visual Studio Code.app 拖入 Applications。
🐧 Linux: 下载 .deb/.rpm 包安装，或使用 Snap: sudo snap install code --classic

❓ Frequently Asked Questions

VS Code 和 Visual Studio 区别？
VS Code 是轻量级编辑器；Visual Studio 是重量级 IDE。大多数开发者用 VS Code 足矣。
VS Code 是免费的吗？
完全免费开源。
