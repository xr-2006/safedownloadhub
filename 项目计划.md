# TulisAja 导航站 — 项目计划

## 定位
解决普通人找不到正版软件官网/下载渠道的问题。用 TulisAja 规则引擎批量生成标准化的导航页内容，吃搜索长尾流量，靠广告/联盟营销变现。

## 市场策略
- **首选英文站**：广告单价高，SEO竞争但可做长尾
- **后续扩展**：马来语站、中文站（TulisAja引擎三语支持）
- **流量关键词模式**：`<软件名> + download / official /官网 / muat turun`

## 内容架构（每软件一页）
```
标题：软件名 + Official Download / Official Site
URL：/steam / /7zip / /obs-studio ...
结构：
  📥 官方下载链接（置顶加粗）
  📝 一句话介绍
  ✅ 正版渠道说明
  📖 安装指南（Win/Mac/Linux）
  ❓ 常见问题
  🔗 官方相关链接
```

## 技术方案
- **静态网站生成器**：Astro（轻量快速，免费部署，SEO友好）
- **部署平台**：Netlify（免费托管 + 自定义域名 + 自动部署）
- **内容引擎**：TulisAja 规则引擎（批量生成标准化内容）
- **域名**：待定
